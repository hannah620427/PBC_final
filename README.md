# 【PBC_final】：智慧演算法排程番茄鐘系統

## 開發環境
本專案環境已透過 `.gitignore` 進行隔離保護，請放心在本地端建立虛擬環境。
(為了避免管理環境用的uv相關檔案push到repository上，增加了這個.gitignore，大家可以不用管這個檔案)

## 套件安裝
請在你的終端機輸入
pip install -r requirements.txt
